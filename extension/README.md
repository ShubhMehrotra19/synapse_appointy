# Synapse Browser Extension

This is the browser extension for Synapse - Second Brain.

## Installation

1. Open Chrome/Edge and navigate to `chrome://extensions/` or `edge://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `extension` folder

## Features

- Capture web content with a single click
- AI-powered content categorization
- Seamless integration with Synapse web app

## Icons

You'll need to add icon files (icon16.png, icon48.png, icon128.png) to the `icons/` folder. For now, you can use placeholder images or create simple icons.
